📝 Mon Blog Django

Bienvenue sur mon premier projet Django 🚀
Il s'agit d'une application web de blog développée avec Python et Django.

🔗 Accès au projet

👉 Lancer en local :

python3 manage.py runserver

Puis ouvrir dans le navigateur :

http://127.0.0.1:8000/

⚙️ Technologies utilisées
Python 3.13
Django 6
HTML / CSS
SQLite (base de données par défaut)

🚀 Installation

Cloner le projet :
git clone https://github.com/ton-username/ton-repo.git
cd ton-repo

Installer les dépendances :
python3 -m pip install -r requirements.txt

Lancer le serveur :
python3 manage.py runserver

📌 Fonctionnalités
Page d’accueil avec affichage dynamique de la date et de l’heure
Interface web personnalisée avec background
Projet structuré avec Django
Accès à l’administration Django
